void LicenseSpring::ExtendedOptions::enableLicenseCorruptionCheck ( bool  enable)

Enable or disable encrypted license data corruption checking.

To prevent writing garbage data to the local license file, this flag decrypts the string and compares it to original data.
This introduces an additional decryption overhead on every license file write.

void LicenseSpring::ExtendedOptions::setCertificateChain ( const std::string &  certificateChain)

Set certificate chain used for verifying LicenseSpring response signatures. This method can be used if you are planning to deploy a self-hosted LicenseSpring instance or a Floating server.

Cert chain should contain a newlines for parsing, and be formatted as --—BEGIN CERTIFICATE--— <Leaf Certificate> --—END CERTIFICATE--— --—BEGIN CERTIFICATE--— <Intermediate Certificate> --—END CERTIFICATE--— --—BEGIN CERTIFICATE--— <Root Certificate> --—END CERTIFICATE--—

Parameters
certificateChainCertificate chain used for signature verification
