◆ getCurrentBundle()

const std::map< std::string, License::ptr_t > & LicenseSpring::BundleManager::getCurrentBundle ( )

Load the local license bundle into memory (lazily).

The first time this is called, it reads the bundle file from disk decrypts it and constructs a map of product-code → License instances. Subsequent calls simply return the cached map.

Returns
A const reference to the map of product code to License::ptr_t.
Exceptions
LocalLicenseExceptionif the bundle file cannot be read or is corrupted.
