In order to compile the project please open WTLSample.sln file with Visual Studio and press F7 (Build Solution).
Solution file is placed in 'build' folder.

Sample application has two screens: activation screen and info screen.

On the activation screen you can enter license key, set path for offline license file or get a trial key.
When you set path for offline license file, if it will be empty, app will ask you to generate request file and check entered keys. 
If both - license key and trial key are empty - it will ask your for getting the trial one.

On info screen there are edit boxes with common info about the license. You can try to update consumtion, check it, deactivate license,
delete license file's folder or open it in explorer, download version list and check for updates.
